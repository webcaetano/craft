# Graphic

Create Phaser Graphic Object

#### `craft.$graphic(options)`

- `options` <Object>
  - `x` <Number> Position x
  - `y` <Number> Position y
  - `group` Phaser group

<!-- protosTpl -->

#### Example:

```javascript
//simple
craft.$graphic();

//full default values
craft.$graphic({
	x:0,
	y:0,
	group:undefined,
})
```
