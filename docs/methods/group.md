# Group

Create Phaser Group

#### `craft.$group(options)`

- `options` <Object>

<!-- protosTpl -->

#### Example:

```javascript
//simple
craft.$group();

//alias 
craft.$g();

//full default values
craft.$group({
	parent:undefined,
	name:undefined,
	addToStage:undefined,
	enableBody:undefined,
	physicsBodyType:undefined
})
```
